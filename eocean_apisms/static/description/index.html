Extensión que permite gestionar campañas de mensajes SMS utilizando la pasarela de servicios de la empresa Entel Ocean

Con esta herramienta, los usuarios pueden enviar mensajes de texto a múltiples contactos a través de la plataforma Odoo de manera eficiente y programada.

Instalación

Para instalar el plugin Touch, siga los siguientes pasos:

  1. Descargue el archivo eocean_apisms.zip desde el sitio oficial de Touch.
  2. Inicie sesión en su instancia de Odoo como administrador.
  3. Diríjase al menu "Aplicaciones" y haga clic en "Cargar módulo".
  4. Seleccione el archivo zip descargado y haga clic en "Cargar".
  5. Una vez cargado el módulo, haga clic en "Instalar/Activar" para habilitar el plugin en su instancia de Odoo.

Uso - Pasarelas

El plugin Touch proporciona una interfaz sencilla y fácil de usar para gestionar las distintas pasarelas dentro de Entel. Para crear una nueva pasarela siga estos pasos:

  1. Diríjase al menu "EOcean - Touch" y seleccione " Pasarelas".
  2. Haga clic en "Nuevo" para crear una nueva pasarela.
  3. Complete los detalles de la pasarela, como el nombre, el ID de Cliente y el Secret ID de envío.
  4. Presione "Guardar" para validar y autenticar al usuario con la pasarela de Entel.

Uso - Campañas SMS

Para crear una nueva campaña de SMS, siga estos pasos:

  1. Diríjase al menu "EOcean - Touch" y seleccione " Campañas SMS".
  2. Haga clic en "Nuevo" para iniciar una nueva campaña.
  3. Complete los detalles de la campaña, como el nombre, el mensaje y la fecha de envío.
  4. Seleccione los contactos a los que desea enviar el mensaje.
  5. Presione "Guardar y Enviar" para comenzar la campaña de SMS.
